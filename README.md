# Origa — AI Automation & Tech Build Studio Portfolio

This repository contains the portfolio website for **Origa — AI Automation & Tech Build Studio**. 

The site is built as a highly polished, single-page responsive web experience with all assets (images/SVGs) fully self-contained/embedded.

## Included Files
- `index.html` (Main entry point for GitHub Pages)
- `origa-portfolio.html` (Original HTML template file)
- `origa-portfolio-desktop.pdf` (Desktop format PDF version of the portfolio)
- `origa-portfolio-landscape.pdf` (Landscape format PDF version of the portfolio)
- `origa-portfolio-portrait.pdf` (Portrait format PDF version of the portfolio)

## How to Deploy to GitHub Pages

To deploy this portfolio to GitHub Pages:

1. **Create a GitHub Repository**: Create a new repository on GitHub (e.g., `origa-portfolio`).
2. **Push the Code**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit of Origa portfolio"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
3. **Enable GitHub Pages**:
   - Go to your repository settings on GitHub.
   - In the left sidebar, click on **Pages**.
   - Under **Build and deployment**, select **Deploy from a branch** as the source.
   - Choose the branch `main` and directory `/ (root)`.
   - Click **Save**.

Your portfolio will be live at `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/` within a few minutes!
